Request Header Parser Microservice

A freeCodeCamp Javascript Back-End Challange

This challange makes use of NodeJS and ExpressJS

Challange:

  • Make use of the data sent in the request.headers method upon accessing a webpage.
  • A JSON object is returned with the parsed data for: IP Address, Language & System Infomation.

Usage:

Click the button below or add /headers in the address bar above.

Output:

{ IP Address: "localhost:8080", Language: "en-US", Operating System: "Windows NT 10.0; Win64; x64" }
